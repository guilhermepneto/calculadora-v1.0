# Calculadora

# Projeto

Projeto desenvolvido durante as primeiras aulas do curso [Academia do Programador](https://www.academiadoprogramador.net/inicio) 2026


# Introdução

Calculadora que roda em console, que realiza as quatro operações básicas, tabuada de qualquer número e que salva e exibe o histórico das operações realizadas.

# Funções

### Operações:

- Soma
- Subtração
- Multiplicação
- Divisão

### Tabuada:

Comando capaz de realizar a tabuada do número desejado, podendo escolher o seu multiplicador máximo.

### Histórico:

Calculadora possui uma memória que armazena uma certa quantidade de operações que quando solicitado pelo usuário, exibe o histórico.

# Como usar a calculadora

1. Clone ou baixe os arquivos do repositório;
2. Abra o emulador de terminal preferido e encontre a pasta raiz do projeto baixado;
3. utilize o comando abaixo para restaurar as dependências do projeto:

    ```
    dotnet restore
    ``` 

4. Compile e execute o projeto com o comando abaixo:
    ```
    dotnet run --project Calculadora.ConsoleApp
    ```

# Requisitos

- .NET 10 SDK
